Information about the project

The project is named "GameSmasher". It is a mashup that combines different APIs in order to show a webpage which give the user a complete view of the videogame he/she is looking for. The user will have the option to sign up in the page and save his/her favorite videogames.

Finally, the mashup will also have a function that shows the game statistics of the user of two specific games: Overwatch and League of Legends.
